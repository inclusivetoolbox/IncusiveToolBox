Configuración

Conecta la caja de herramientas con tu Google Sheet y configura el acceso del profesor.

Paso 1 — Google Sheet (lectura):
Archivo → Compartir → Publicar en la web → Hoja1 → CSV → Publicar. Copia la URL.

Paso 2 — Apps Script (agregar y eliminar):
En tu Sheet: Extensiones → Apps Script → pega este código y publica como app web:

function doPost(e) { var sheet = SpreadsheetApp.getActiveSheet(); var d = JSON.parse(e.postData.contents); if (d.action === 'delete') { sheet.deleteRow(parseInt(d.row)); } else { sheet.appendRow([d.modulo,d.titulo,d.tipo,d.link,d.descripcion]); } return ContentService.createTextOutput('ok'); }
Al publicar: Ejecutar como: Yo · Acceso: Cualquier persona. Copia la URL.
